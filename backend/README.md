2. lista przepisów
3. jak bedziesz w przepisie to składniki osobno i osobno treść przepisu

5. Osobno moje przepisy i tablice przepisów do których mozna sobie zapisać:
   moje przepisy->kategorie->lista przepisów
   moje tablice-><nazwy tablic>->zapisane przepisy
6. najlepsze przepisy z ostatnich 30 dni / 365 dni / all time + wyszukiwanie

   - tagi z liczbą od najczestszych do najmniej
